# OfficeManagementSystem
It is a java project foe management of office. 

It contains two login:

1) Employee login
2) Admin login

Employee Login:

1) A employee can login and give details of to the manager.
2) Apply for leave.
3) Get circulars from Admin.

Admin login:

1) Add new employ.
2) transfer circular
3) Remove an employ
4) Grant leave
